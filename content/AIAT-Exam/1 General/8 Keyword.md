In Python, **keywords** are reserved words that have a special predefined meaning and cannot be used as identifiers (e.g., variable names, function names). Keywords are the building blocks of Python syntax and structure. 

Most keywords are explained on a different page, so the "Context" column will link to the details.

| Context                                                                    | Keyword    | Description                                                                                                                                                                                    |
| -------------------------------------------------------------------------- | ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[AIAT-Exam/1 General/5 Datatype#Boolean\|Datatype]]             | `False`    | Boolean value representing False.                                                                                                                                                              |
| [[AIAT-Exam/1 General/5 Datatype#Boolean\|Datatype]]             | `True`     | Boolean value representing True.                                                                                                                                                               |
| [[AIAT-Exam/1 General/5 Datatype#None\|Datatype]]                | `None`     | Represents the absence of a value or a null value.                                                                                                                                             |
| [[AIAT-Exam/1 General/7 Operator#Logical Operators\|Operator]]   | `and`      | Logical AND operator. True if all operands are True.                                                                                                                                           |
| [[AIAT-Exam/1 General/7 Operator#Logical Operators\|Operator]]   | `or`       | Logical OR operator. True if at least one operand is True.                                                                                                                                     |
| [[AIAT-Exam/1 General/7 Operator#Logical Operators\|Operator]]   | `not`      | Logical NOT operator. Negates a boolean value.                                                                                                                                                 |
| [[AIAT-Exam/1 General/7 Operator#Membership Checking\|Operator]] | `in`       | Checks if an element exists in a sequence or collection. Also used to iterate a variable in a for loop.                                                                                        |

| [[AIAT-Exam/2 Control Flow/5 Function#Scope\|Variable]]          | `global`   | Declares a variable as global, accessible outside the current scope.                                                                                                                           |

| [[AIAT-Exam/2 Control Flow/2 Conditions\|Flow Control]]          | `else`     | Defines the block of code to execute if the `if` condition is False. Also has a special use-case in the context of [[AIAT-Exam/2 Control Flow/3 While Loop\|while]], for and except. |
| [[AIAT-Exam/2 Control Flow/3 While Loop\|Flow Control]]          | `while`    | Defines a loop that continues as long as the condition is True.                                                                                                                                |

| [[AIAT-Exam/2 Control Flow/3 While Loop#Continue\|Flow Control]] | `continue` | Skips the remaining code in a loop iteration and moves to the next iteration.                                                                                                                  |
| [[AIAT-Exam/2 Control Flow/5 Function#Basic\|Function]]          | `def`      | Defines a new function or method.                                                                                                                                                              |
| [[AIAT-Exam/2 Control Flow/5 Function#Basic\|Function]]          | `return`   | Exits a function and optionally passes back a value.                                                                                                                                           |
