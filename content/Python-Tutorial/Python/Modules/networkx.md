__tools for the study of the structure and dynamics of complex networks__

|Method|Description|
|---|---|
|`nx.Graph()`|Create an undirected graph|
|`nx.DiGraph()`|Create a directed graph|
|`G.add_node(node)`|Add a node to the graph|
|`G.add_nodes_from(nodes)`|Add multiple nodes to the graph|
|`G.add_edge(u, v)`|Add an edge between nodes u and v|
|`G.add_edges_from(edges)`|Add multiple edges to the graph|
|`G.number_of_nodes()`|Get the number of nodes in the graph|
|`G.number_of_edges()`|Get the number of edges in the graph|
|`G.nodes()`|Get a list of nodes in the graph|
|`G.edges()`|Get a list of edges in the graph|
|`G.neighbors(node)`|Get the neighbors of a node|
|`G.degree(node)`|Get the degree of a node|
|`nx.shortest_path(G, u, v)`|Find the shortest path between nodes u and v|
|`nx.connected_components(G)`|Find the connected components of the graph|
|`nx.draw(G)`|Draw the graph using Matplotlib|
