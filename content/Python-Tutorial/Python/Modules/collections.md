|Class/Function|Description|
|---|---|
|`deque()`|Double-ended queue implementation.|
|`ChainMap()`|Dictionary-like class for creating a single view of multiple mappings.|
|`Counter()`|Dictionary subclass for counting hashable objects.|
|`OrderedDict()`|Dictionary subclass that remembers the order of entries.|
|`defaultdict()`|Dictionary subclass that calls a factory function to provide missing values.|

|`UserList()`|Wrapper around list objects for easier list subclassing.|

|`deque.extend(iterable)`|Extend the deque by appending elements from the iterable to the right end.|
|`deque.extendleft(iterable)`|Extend the deque by appending elements from the iterable to the left end.|
