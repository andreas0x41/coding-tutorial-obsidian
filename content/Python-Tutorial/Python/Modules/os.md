__functions for interacting with the os, like file operations, environment variables, directory operations, ...__

| Method/Attribute | Description |
|---|---|
| `os.name` | Name of the operating system (e.g., 'posix', 'nt') |
| `os.getcwd()` | Get current working directory |
| `os.chdir(path)` | Change current working directory |
| `os.listdir(path='.')` | Get list of files and directories in a directory |
| `os.mkdir(path)` | Create a directory |
| `os.makedirs(path)` | Create a directory, including any necessary parent directories |
| `os.remove(path)` | Remove (delete) a file |
| `os.rmdir(path)` | Remove (delete) a directory |
| `os.rename(src, dst)` | Rename a file or directory |
| `os.path.exists(path)` | Check if a path exists |
| `os.path.isfile(path)` | Check if a path is a regular file |
| `os.path.isdir(path)` | Check if a path is a directory |
| `os.system(command)` | Execute a shell command |
