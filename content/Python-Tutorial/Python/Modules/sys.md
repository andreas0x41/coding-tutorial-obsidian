__variables used or maintained by the interpreter, functions that interact with the Python runtime environment__

|Attribute/Method|Description|
|---|---|
|`sys.argv`|Command-line arguments passed to the script|
|`sys.exit([code])`|Exit from the Python interpreter with an optional exit code|
|`sys.version`|Python interpreter version information|
|`sys.path`|List of directories where Python looks for module files|
|`sys.platform`|Operating system platform identifier|
|`sys.stdout`|Standard output stream (used for printing)|
|`sys.stdin`|Standard input stream (used for reading)|
|`sys.stderr`|Standard error stream (used for printing error messages)|
|`sys.modules`|Dictionary of loaded modules|
|`sys.getsizeof()`|Get the size of an object in bytes|
|`sys.setrecursionlimit()`|Set the maximum recursion depth|
|`sys.getrecursionlimit()`|Get the current recursion limit|
|`sys.getdefaultencoding()`|Get the default string encoding used by Python|
|`sys.settrace()`|Set a function to be called on every function call|
|`sys.gettrace()`|Get the current trace function|
|`sys.exc_info()`|Get information about the current exception|
